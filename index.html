Geodesic

Geodesic is a Sass toolkit for making large systems out of small parts. It provides lots of discrete chunks powered by configurable fiddly bits. Clone Geodesic, configure, and then start adding and deleting as you see fit. You’ll have a complete functional CSS system in no time!

The Grid

Geodesic ships with an 8 column grid that breaks down to a 4 column grid on small screens. But that’s crazy-easy to change just by changing the values in settings/_grids.scss. All grid classes are generated from these values, so you can switch some numbers up and get right down to business.
